# Advanced Real Tool Demos

These demos are optional and intended for advanced setups.

Rules:

1. Run with `--mode autonomous_pty`.
2. Record metadata in a manifest (`*.manifest.yaml`).
3. Include tool version/date when using vendor naming in public docs.

Example:

```bash
studio run examples/real/real_agent_demo.yaml --mode autonomous_pty --output-dir outputs
```
